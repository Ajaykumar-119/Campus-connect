# CampusConnect – College Social Network Platform

CampusConnect is a full-stack web application designed to improve communication and collaboration among students within a college. The platform provides a centralized space where students can connect, share updates, participate in discussions, and stay informed about college activities.

## Features

- User Registration & Login
- Secure Authentication
- Student Profiles
- Create, Edit, and Delete Posts
- Like and Comment on Posts
- News Feed
- Responsive User Interface
- College Community Interaction
- Database Integration
- Session Management

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/CampusConnect-College-Social-Network.git
```

### Navigate to the Project

```bash
cd CampusConnect-College-Social-Network
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
