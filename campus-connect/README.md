# Campus Connect

Campus Connect is a college project source-code package for a student community portal. It includes a Node.js backend, local JSON data storage, and a responsive browser frontend.

## Features

- Student/faculty/admin profile simulation
- Dashboard with live counts
- Notice board with new notice posting
- Event listing with event creation
- Campus clubs directory
- Lost and found reporting
- Local JSON-backed data storage

## How to run

1. Install Node.js if it is not already installed.
2. Open this folder in a terminal.
3. Run: npm start
4. Open: http://localhost:3000
